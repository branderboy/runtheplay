The Run the Play Flywheel
1. Run the Play discovers the podcast
You create a basic public profile using:
Podcast name
Hosts
YouTube and social links
Public reach
Recent episodes
Categories
Available public contact information
The profile starts as unclaimed.
2. Run the Play features the podcast
The show appears in:
Viral clips
Weekly charts
Growth rankings
Underrated-show spotlights
Audience breakdowns
Example advertising plays
Budget-based media plans
The content gives the podcast visibility before it ever pays.
3. The podcast receives an alert
Send a direct message or email such as:
Your podcast was featured by Run the Play. We created a public profile showing your reach and platforms. Claim it free to correct your information, add your advertising opportunities, and control how brands contact you.
The alert should include:
Link to the social post
Link to the podcast profile
Current public metrics
Claim Profile button
4. The podcast claims its profile
Once verified, the owner can:
Correct show information
Connect YouTube and social accounts
Add audience demographics
Upload a media kit
Add rates or contact-for-pricing
Add available advertising formats
Set the preferred inquiry method
Submit clips for possible features
5. The profile improves the planner
The more complete the profile becomes, the more accurately Run the Play can include it in:
Ad Planner recommendations
Campaign baskets
Audience searches
Budget filters
Example plays
Weekly availability alerts
6. Better profiles create better content
The verified information gives Run the Play stronger material for:
New clips
Verified stat bars
Growth reports
Advertising-opportunity posts
Podcast comparisons
“What this budget can buy” content
That content attracts more buyers and more media owners.
